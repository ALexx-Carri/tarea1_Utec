# tarea1_Utec
Esta es la tarea que nos dejaron en la semana 2 de la clase de programación 3
Nuestro programa usa Arrays dinamicos debido a que son la mejor opción para ahorrar memoria y por que es el meotodo más proximo a las referencias que nos dieron de NumPy y PyTorch que trabajan con vectores
Hemos ido columna por columna, se llena cada columna de 0 y va bajando hasta el limite de la columna, yendo progresivamente columna por columna hasta terminar con las misma y por consecuente rellena las filas tambien
